### Crear un nuevo proyecto Angular
```
ng new nombre-proyecto
```
### Servir la aplicación en desarrollo
```
ng serve
```
### Alias corto
```
ng s
```
### Compilar para producción
```
ng build --configuration production
```
### Alias corto
```
ng build --prod
```
### Ejecutar pruebas unitarias
```
ng test
```
### Ejecutar pruebas end-to-end
```
ng e2e
```
### Mostrar la versión de Angular CLI y dependencias
```
ng version
```
### Generar un componente
```
ng generate component nombre-componente
```
### Alias corto
```
ng g c nombre-componente
```
### Generar un servicio
```
ng generate service nombre-servicio
```
### Alias corto
```
ng g s nombre-servicio
```
### Generar un módulo
```
ng generate module nombre-modulo
```
### Alias corto
```
ng g m nombre-modulo
```
### Generar una directiva
```
ng generate directive nombre-directiva
```
### Alias corto
```
ng g d nombre-directiva
```
### Generar un pipe
```
ng generate pipe nombre-pipe
```
### Alias corto
```
ng g p nombre-pipe
```
### Instalar Angular Material
```
ng add @angular/material
```
### Instalar PWA
```
ng add @angular/pwa
```
### Instalar librerías externas
```
ng add nombre-paquete
```
### Generar un módulo con rutas
```
ng g m nombre-modulo --routing
```
### Generar un guard (protección de rutas)
```
ng g guard nombre-guard
```
### Compilar con optimización y análisis
```
ng build --configuration production --stats-json
```
### Analizar el tamaño del bundle
```
npm run analyze
```
